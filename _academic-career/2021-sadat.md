## Role Description

Designed and delivered comprehensive web design curriculum to high school students at Sadat School for Exceptional Talents, focusing on practical, hands-on programming skills and real-world project development during Fall 2021 semester.

## Key Achievements

- Developed and taught intensive web design course covering HTML5, CSS3, and JavaScript fundamentals
- Mentored 30+ high school students in creating their first functional websites and web applications
- Implemented project-based learning approach where students built portfolio websites and interactive web pages
- Created engaging coding exercises and real-world challenges to enhance practical programming skills
- Provided individual coaching to help students troubleshoot code and debug their projects
- Introduced responsive design principles and modern web development best practices
- Inspired next generation of developers through interactive coding sessions and creative web projects
- Received excellent feedback from students and school administration for teaching methodology
